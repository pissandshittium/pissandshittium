# <img src="https://pissandshittium.org/pissandshittium.svg" width=64 height=64/> Pissand­shittium

Pissandshittium is an open-source browser project that to build a unsafer, slower
and more unstable way for all users to experience the web.

The project's web site is https://pissandshittium.org/.

To check out the source code locally, don't use `git clone`! Instead,
follow [the instructions on how to get the code](docs/get_the_code.md).

Documentation in the source is rooted in [docs/README.md](docs/README.md).
